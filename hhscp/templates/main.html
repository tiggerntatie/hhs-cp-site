Introduction

Computer Programming is a one-semester course that will introduce the basic techniques of computer programming using the Python programming language. Python is a modern, interpreted language that is relatively easy to learn, while possessing an astonishing depth and flexibility that make it suitable for sophisticated applications in business, arts, entertainment, science and engineering.

In the first few weeks of this class you will learn the tools and basic skills needed to write simple programs in Python. Each week will develop several new skills or techniques that may be required to complete a series of challenge assignments. Some of the skills you will learn during this period include:

  • Interactive mode: expressions, types, variables, collections, dir(), help()
  • Console I/O
  • Lambda functions
  • List comprehensions
  • Objects, properties, methods and inheritance
  • Conditionals and control flow statements
  • Functions

Later in the course you will plan and execute a series of one to two-week long mini-projects

I Provide You With:

  • Computers with Internet access.
  • Python 2.6
  • Python Integrated Development Environment: PyCharm
  • Git version control
  • Pygame Python game development library.

If you want to work outside of class:

  • We will be using Python 2.6 during this course. This is not the latest version that you will find available for download. You can probably use version 2.7, but do not install version 3.0!
  • A USB Flash drive may come in handy for transferring files.
  • You may install the latest version of the PyCharm Python IDE on your personal computer. We have a classroom license that entitles you to install and use PyCharm at home while you are enrolled in the course.
  • Twisted and Pygame will be used later in the course, but are not required initially.

I Expect YOU To:

  • Come to class.
  • Be on time. Please let me know if you expect to be late.
  • Be responsible. If you are absent on a day when work is due I will expect you to have it ready the day you return. If you know you will not be able to take a test or quiz at the scheduled time make arrangements with me (by e-mail, in person, etc.) and receive confirmation before the scheduled class.
  • Always treat others with kindness and respect.

Textbook

There isn't one. I will have a couple of reference books on hand during class, but the best sources are always on the Internet. I expect you to use the Internet as a tool for solving problems and learning new techniques.

X Periods

This class has an X period on Thursday. Please check in at the start of class (or make arrangements with me in advance). I do take and submit attendance for the X! The X is available to you for finishing your work, getting help or completing missed quizzes.

Classwork, Quizzes and Project Grading
Weekly Challenges At the beginning of the semester, each week will bring a series of challenge assignments that you must complete and upload to a web site. Challenges must typically be completed on Friday. Scoring rubric:
  • 0/10 - Not submitted
  • 4/10 - Submitted, but does not run without errors.
  • 6/10 - Submitted, runs without error, but does not meet objectives.
  • 10/10 - All objectives met
Quizzes Every two weeks or so, you will be quizzed on some of the techniques used in the challenge assignments. Quizzes will usually consist of creating a short program during class and will be “open Internet”. Each quiz will list several requirements and each requirement is scored as one or two points.
Project Reviews Your final project will require planning. You or your team must develop a plan and demonstrate weekly progress towards completion. A well-designed and implemented application represents a substantial amount of work!
Mini Projects After the initial introductions you will work on some more substantial projects of your own choosing. For example, graphics programming or game development with Pygame.
Final Project This longer project will be the “final exam” for the course.
Late Work All work is subject to being graded once the due date has arrived. Work that is completed or revised after the due date, may be awarded full credit within two weeks of the original due date. You must inform me by e-mail of updated pages or any time you submit late work, as I have no way of knowing what you want me to look at!
Grade Weighting
1st Quarter (emphasizing challenges) Challenges 60%, Quizzes 20%, Mini Projects 20%
2nd Quarter (emphasizing projects) Challenges 30%, Project Weekly Reports 30%, Mini Projects 40%.
Semester Final Project is worth 20% of the semester grade.
