The purpose of this challenge is to gain proficiency with exception handling code.

Write and submit a Python program that computes areas of several different types of polygons: squares, rectangles, triangles, parallelograms, trapezoids and kites. Areas should be displayed with six significant figures (at most). If the user enters an unknown type of polygon, the program should respond that it doesn't know how to find that (see example). If the user enters a dimension that is not a valid number, the program should respond that it is not valid length and invite the user to try again.

This challenge is functionally similar to Calculating Areas with Class, however you must add exception handling to catch data entry errors.
