The purpose of this challenge is to practice building a simple console game using a game tree.

Your task is to create a program using Python that plays the Roman Board Game - Merels. The article describes the rules and let's you play the game against yourself.

You may write a game playing program for another variation of Merels. Refer to the Internet for rules, etc.

Your finished program should work in a way that is similar to the example Tic Tac Toe game. In fact, the TTTPosition class may be used as a base class for another program that plays the ancient Roman version of Merels. If you choose this route, you will find that very few changes are necessary!

An example session playing the ancient Roman version of Merels follows. Notice that the circular grid is represented as a 3x3 square grid, just like the Tic Tac Toe board. The game allows you to place three pieces on the board, then forces you to slide them in any direction (but only by one space). You win by getting three in a row through the center only.

Some basic requirements for your submission:

  • You may use the code supplied in the Simple Games as a starting point ( consoletictactoe.py).
  • As always, you may mine the Internet for ideas and inspiration, but you may not submit someone else's implementation of any Merels game (I don't think there are any implementations of the ancient Roman version).
  • Please credit any sources for inspiration or code snippets.

My “reference” implementation (which I certainly don't claim to be optimal) has 147 lines (including blanks and extensive comments), and imports consoletictactoe so that it can use the base class TTTPosition.

Note: your submitted program will not be automatically checked for correctness. Any submission will automatically “pass”. I will download and execute your submitted scripts, however.
