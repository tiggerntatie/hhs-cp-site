The purpose of this challenge is to practice building a simple Pygame game using a game tree.

Your task is to create a program using Python and Pygame that plays the Roman Board Game - Merels. The article describes the rules and let's you play the game against yourself.

You should use or reuse your submission for p03 Console Merels as a basis for completing this project. You should also refer to the tutorial on constructing a Graphical Tic Tac Toe game.

You may write a game playing program for another variation of Merels. Refer to the Internet for rules, etc.

Some basic requirements for your submission:

  • You should allow the user to choose who will play the first move.
  • You do not have to use background images with your program. If images are required, then you will have to e-mail them to me or submit everything in a single zip file (unless I can modify the system to allow multiple submitted files).
  • You may use the code supplied in the Graphical Tic Tac Toe as a starting point ( pygametictactoe.py).
  • As always, you may mine the Internet for ideas and inspiration, but you may not submit someone else's implementation of any Merels game (I don't think there are any implementations of the ancient Roman version).
  • Please credit any sources for inspiration or code snippets.

Note: your submitted program will not be automatically checked for correctness. Any submission will automatically “pass”. I will download and execute your submitted scripts, however.
