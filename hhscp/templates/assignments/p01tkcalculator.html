The purpose of this challenge is to practice building a GUI application with Python.

Write and submit a Python program that simulates an ordinary four function calculator, using the Tkinter GUI tools in Python.

Your calculator should look something like this:

Some basic requirements for your calculator:

  • Calculator starts with 0 in the display
  • Pressing a number/decimal key while 0 is in the display will replace the zero with the new number
  • Supports decimal numbers (i.e. floating point)
  • Does not allow more than one decimal point (obviously!)
  • Pressing a number/decimal key while the result of the last operation is displayed will replace the old number with the new number
  • Supports using both on-screen buttons and keyboard keys
  • May support additional “features” beyond what is described here

Note: your submitted program will not be automatically checked for correctness. Any submission will automatically pass. I will download and execute your submitted scripts, however.
