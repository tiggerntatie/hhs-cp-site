The purpose of this challenge is to practice using conditionals (decision making) with loops.

Write and submit a Python program that computes areas of several different types of polygons: squares, rectangles, triangles, parallelograms, trapezoids and kites. Areas should be displayed with six significant figures (at most). If the user enters an unknown type of polygon, the program should respond that it doesn't know how to find that (see example).

Output of your program should look like this:

Following are area formulas for these polygons (in case you don't know or can't remember)

  • Square: \(A = s^2\) where A is area and s is the length of one side
  • Rectangle: \(A = b h\) where A is area, b is the base length and h is the height
  • Parallelogram: \(A = b h\) where A is area, b is the base length and h is the height
  • Triangle: \(A = \frac{1}{2} b h\) where A is area, b is the base length and h is the height
  • Trapezoid: \(A = \frac{b_1 + b_2}{2} h\) where A is area, b1 and b2 are the base lengths and h is the height
  • Kite: \(A = \frac{d_1 d_2}{2}\) where A is area, d1 and d2 are the diagonal lengths
